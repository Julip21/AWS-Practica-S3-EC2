# AWS Cloud Project - EC2 + S3 Architecture

## 🚀 Overview
This project showcases the design and implementation of a basic cloud architecture on AWS, combining compute and storage services while applying security best practices.

It simulates a real-world scenario where a web server is deployed and integrated with cloud storage.

## 🏗️ Architecture
The solution includes:

- Amazon EC2 for compute resources  
- Amazon S3 for object storage  
- Security Groups for traffic control  
- IAM for identity and access management  

## ⚙️ Implementation
During this project, the following tasks were completed:

- Deployed and configured an EC2 instance with secure SSH access  
- Created and managed S3 buckets for storage  
- Configured Security Groups to allow controlled inbound traffic (SSH/HTTP)  
- Implemented IAM roles and policies following the principle of least privilege  
- Verified connectivity and interaction between services  

## 🔐 Security Best Practices
- Restricted access to specific ports (22, 80)  
- Used SSH key pairs for secure authentication  
- Applied least privilege principles in IAM  
- Implemented basic network access control  

## 📊 Results
- Successfully deployed a running EC2 instance accessible via SSH  
- Configured a secure S3 bucket for storage  
- Established controlled and secure communication between AWS services
